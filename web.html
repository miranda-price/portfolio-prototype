  • Final Product
  • Responsive Webpages

Final Product

This final project took four separate biographical webpages about impactful women and combined them into one cohesive project. The project had three main goals: content integration, webpage responsiveness, and search engine optimization. For content integration, the challenge was to adapt the content from four webpages from four different creators so that the information architecture and styling of each page remained consistent. Some pages had their content reordered to fit the outline of the other pages or to fit the layout structure of the other pages. HTML had to be edited so that the styling from one of the original sites would work on all the pages of the new site. Webpage responsiveness and search engine optimization are detailed below.

The top of the homepage is pictured to the right.

Responsive Webpages

This project emphasized creating webpages that were responsive and that worked well in mobile browsers. Breakpoints were used to ensure that images, tables, and side-by-side content translated well to a smaller device without creating a horizontal scroll bar. Readability and usability were still emphasized for this project, even when working with smaller screens.

The two photos to the right demonstrate the same part of the website displayed on a laptop and on a smart phone.

Search Engine Optimization

This project also emphasized optimizing the site for search engines to give the site more visibility on the Internet. For this HTML, search engine optimization meant including title tags in the head element, using HTML elements correctly, outlining and ordering HTML elements correclty, and writing descriptive alts for all images and hyperlinks. I also setup and incorporated a sitemap file and a Google Analytics for this site to track the site's user traffic.
